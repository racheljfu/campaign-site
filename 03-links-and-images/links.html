This page is about links. There are three types of links.

  • Absolute links, which you should generally use to direct users to a different website. For example, Mozilla Developer Network, is a very good resource for web developers.
  • Relative links are best for pointing to another file in your website that are in the same or deeper folder, like to our extras page.
  • Root-relative links are relative to the "root" of the whole site, like to the home page of our website, but those aren't useful to us right now.
