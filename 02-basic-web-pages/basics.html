Lists

This is how you make an unordered list.
  • First add a "ul" element.
  • Add each item in its own "li" element
  • They don't have to be in a certain order
This is how to make an ordered list.
  1. Notice the "ol" element wrapping everything
  2. But the list item elements are the same.
  3. The numbers increment on their own.
  4. You should be noticing things in this precise order.
